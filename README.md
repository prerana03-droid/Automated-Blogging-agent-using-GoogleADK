# Automated-Blogging-agent-using-GoogleADK
Automated blogging system that uses AI agents to generate comprehensive blog posts with AI-generated images and publish them to Blogger. The system leverages Google's Agent Development Kit (ADK), Gemini AI models, DALL-E 3 for image generation, and the Blogger API

•	Takes a topic as input
•	Researches it using Google Search
•	Writes a 1000-word article with Gemini AI
•	Creates 3-5 custom images with DALL-E 3
•	Formats everything professionally with HTML/CSS
•	Publishes to Blogger as a draft so I can review before final post
